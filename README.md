  - Pêche dans les océans profonds

  - Pêche dans les biomes froids
  - Pêche dans les biomes mushrooms
  - Pêche dans les déserts
  - Pêche dans les marais
  - Pêche dans les mesas

  - Dragon Egg

 - Ajout d'avancements
  - Avancements exclusifs à PopolWorld
